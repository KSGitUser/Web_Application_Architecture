Цель  задания — научиться применять архитектурные паттерны.
1. Найти и указать в проекте Front Controller и расписать классы, которые с ним взаимодействуют.
    Class Request  в vendor\symfony\http-foundation\Request.php
    с ним взаимодействует Kernel и контроллеры в проекте
2. Найти в проекте паттерн Registry и объяснить, почему он был применён.
    class Container находится в vendor\symfony\dependency-injection\Container.php
    Используеся для добавления и получения конфигурационных данных проекта
3. Разобраться и объяснить, какой тип модели используется в проекте.
4. Переписать проект, используя паттерн Page Controller.
